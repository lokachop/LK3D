Module proctex

Procedural texturing

Module that generates simple procedural textures
Currently its very primitive and not good at all, it hasn't been touched in a while
The ProcTex table will probably be deprecated soon aswell

LK3D.ProcTex.New(name, w, h)
Declares a new procedural texture
Secretly an alias to LK3D.DeclareTextureFromFunc

Parameters:

  • name string LK3D texture name
  • w number Texture width
  • h number Texture height

Usage:

    LK3D.ProcTex.New("proc_something", 256, 256)
LK3D.ProcTex.ApplySolid(name, col_r, g, b)
Applies a solid colour to a procedural texture

Parameters:

  • name string LK3D texture name
  • col_r color Color object or red channel value
  • g number Green channel value
  • b number Blue channel value

Usage:

  • -- with colour object
 LK3D.ProcTex.ApplySolid("proc_something", Color(128, 255, 128))
  • -- with RGB value
 LK3D.ProcTex.ApplySolid("proc_something", 32, 64, 96)

LK3D.ProcTex.Operator(name, call)
Sets the colour of each pixel based on a lua function

Parameters:

  • name string LK3D texture name
  • call function Function to use, refer to usage

Usage:

    LK3D.ProcTex.Operator("proc_something", function(xc, yc, w, h) -- generic gradient
  return Color((xc / w) * 255, (yc / h) * 255, 0)
end)

LK3D.ProcTex.PerlinMask(from, to, sx, sy, treshold, seed)
Masks a texture onto the texture using perlin noise

Parameters:

  • from string Texture to be masked
  • to string Texture to be applied onto
  • sx number X scale of the perlin noise
  • sy number Y scale of the perlin noise
  • treshold number Threshold to mask from, not a hard cut
  • seed number Seed for perlin noise

Usage:

    LK3D.ProcTex.LK3D.ProcTex.PerlinMask("blend_orange", "blend_show", 2, 2, .45, 637742)
