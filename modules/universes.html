Module universes

Universe Module

This module handles the universe system in LK3D
Universes are where all of the objects, lights, particles and physics bodies live in LK3D

Functions

LK3D.NewUniverse(tag) Creates a new universe
LK3D.PushUniverse(univ) Pushes a new active universe to the stack
LK3D.PopUniverse() Restores the last active universe from the stack
LK3D.WipeUniverse(univ) Clears the active universe of all of its contents
LK3D.GetUniverseByTag(tag) Gets a universe by its tag
LK3D.GetUniverseObjects(tag) Gets the list of objects from the active universe

LK3D.GetUniverseObjects(tag)

WARNING

The returned table is internal and modifying it might lead to crashes!

Gets the list of objects from the active universe

Parameters:

  • tag optional string Universe tag

Returns:

    table Object list

Usage:

    LK3D.PushUniverse(universe_imaginary)
  local objects = LK3D.GetUniverseObjects()
  for k, v in pairs(objects) do
    print("Object named \"" .. k .. "\"!")
  end
