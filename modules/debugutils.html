Module debugutils

Helpful debug render functions

You can think of this as LK3D's version of debugoverlay
These only work with LK3D.Debug set to true!

Functions

Functions

LK3D.DebugUtils.Line(start, endpos, life, col)
Draws a debug line

Parameters:

  • start vector Start position of the line
  • endpos vector End position of the line
  • life number How long should the line be rendered for (in seconds)
  • col color Colour of the line

Usage:

    -- draws a red vertical line that persists for 16s
 LK3D.DebugUtils.Line(Vector(0, 0, 0), Vector(0, 0, 1), 16, Color(255, 0, 0))
LK3D.DebugUtils.Cross(pos, size, life, col)
Draws a debug cross

Parameters:

  • pos vector Center of the cross
  • size number Size of the cross
  • life number How long should the cross be rendered for (in seconds)
  • col color Colour of the cross

Usage:

    -- draws a red cross at the center of the universe that persists for 16s
 LK3D.DebugUtils.Cross(Vector(0, 0, 0), 1, 16, Color(255, 0, 0))
LK3D.DebugUtils.Box(pos, size, ang, life, col)
Draws a debug box

Parameters:

  • pos vector Center of the box
  • size vector Scale of the box
  • ang angle Angle of the box
  • life number How long should the box be rendered for (in seconds)
  • col color Colour of the box

Usage:

    -- draws a red box at the center of the universe, rotated 45 degrees that persists for 16s
 LK3D.DebugUtils.Box(Vector(0, 0, 0), Vector(1, 1, 1), Angle(45, 0, 0), 16, Color(255, 0, 0))
