Module radiosity

Lightmapping / Radiosity

Module that generates calculates lightmaps on objects using radiosity
Will probably soon be rewritten due to lots flaws and unmantainable code
This module is basically a GLua implementation of this radiosity article
Reading the manual entry on the lightmapper is recommended!

Functions

LK3D.SetLightmapped(object) Marks an object to be lightmapped
LK3D.CommitLightmapping() Runs the lightmapper
LK3D.ClearLightmapCache() Clears the PNG lightmap cache
LK3D.LoadLightmapFromFile(obj_idx) Loads a lightmap of an object from a file in LKPack
LK3D.ExportLightmaps() Exports all of the lightmaps on the active universe

Parameters

LK3D.LIGHTMAP_RES To-compute lightmap resolution
LK3D.LIGHTMAP_AUTO_EXPORT Automatically export lightmaps
LK3D.RADIOSITY_DO_RT Raytrace lighting rather than radiosity (broken)
LK3D.RADIOSITY_STEPS Radiosity step count
LK3D.RADIOSITY_BUFFER_SZ Radiosity internal capture buffer size
LK3D.RADIOSITY_FOV Radiosity camera FOV, leave at 90
LK3D.RADIOSITY_LIGHTSCL_DIV Radiosity light object scale division constant
LK3D.RADIOSITY_REFLECTANCE Radiosity patch reflectance
LK3D.RADIOSITY_MUL_EMMISIVE_START Radiosity emmisive surface multiplier
LK3D.RADIOSITY_MUL_RENDER Radiosity end-lighting multiplier
