Module trace

Trace (Raycast) Module

This module handles traces in LK3D, Which allow you to perform raycasts
It currently only supports rays

LK3D.TraceRayObject(pos, dir, name, bfcull)
Traces a ray to a specific object

Parameters:

  • pos vector Ray origin
  • dir vector Ray direction
  • name string LK3D Object index tag to raycast
  • bfcull optional bool Whether we should cull backfaces

Returns:

  1. vector Hit position
  2. vector Hit normal
  3. vector Distance
  4. table UV coords
  5. number Triangle index

Usage:

    local pos, norm, dist = LK3D.TraceRayObject(Vector(0, 0, 4), Vector(0, 0, -1), "cube_center", true)
LK3D.TraceRayScene(pos, dir, bfcull, dist)
Traces the entire scene

Parameters:

  • pos vector Ray origin
  • dir vector Ray direction
  • bfcull optional bool Whether we should cull backfaces
  • dist optional number Ray distance

Returns:

  1. vector Hit position or table if LK3D.TraceReturnTable is true
  2. vector Hit normal
  3. vector Distance
  4. table UV coords
  5. number Triangle index

Usage:

  • local pos, norm, dist = LK3D.TraceRayScene(Vector(0, 0, 4), Vector(0, 0, -1), true, 512000)
  • LK3D.SetTraceReturnTable(true)
local tbl = LK3D.TraceRayScene(Vector(0, 0, 4), Vector(0, 0, -1), true, 512000)
LK3D.SetTraceReturnTable(false)
local objTag = tbl.obj
local dist = tbl.dist
local pos = tbl.pos
local norm = tbl.norm
local uv = tbl.uv
local tri = tbl.tri
