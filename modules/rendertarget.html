LK3D.RTToMaterial(rt, transp, ignorez)
Turns a rendertarget into a lua material
Not an LK3D Texture

Parameters:

  • rt rendertarget Rendertarget to use
  • transp optional bool Add $alphatest shader parameter
  • ignorez optional bool Add $ignorez and $nocull shader parameter

Returns:

  1. material Material of the RT
  2. material Lightmapped material of the RT

Usage:

    local rtMat = LK3D.RTToMaterial(rt_render)
LK3D.RTToMaterialEx(rt, params)
Extended version of LK3D.RTToMaterial()
Not an LK3D Texture

Parameters:

  • rt rendertarget Rendertarget to use
  • params table Parameters, refer to usage

Returns:

  1. material Material of the RT
  2. material Lightmapped material of the RT

Usage:

    local rtMat = LK3D.RTToMaterialEx(rt_render, {
  ["nocull"] = false,
  ["ignorez"] = false,
  ["vertexcolor"] = true,
  ["vertexalpha"] = true,
  ["alphatest"] = false,
})

LK3D.RenderQuick(call)
Calls a function to be drawn on the active rendertarget

Parameters:

  • call function Function to render stuff

Usage:

    LK3D.RenderQuick(function()

  -- ScrW() and ScrH() are the RT size
  surface.DrawRect(0, 0, ScrW() * .5, ScrH() * .5)
end)
