Module lkpack

LK3D Data Archive

Module to handle LKPack, a method to pack data into a single .ain file for workshop
It is highly recommended to read through the Basics of LKPack manual first before using these functions.

Functions

LK3D.MakeLKPack(dir)
Creates an LKPack file from a dev directory

Parameters:

  • dir string Dev directory to turn into an LKPack

Usage:

    LK3D.MakeLKPack("deepdive_content")
LK3D.LoadLKPack(name)
Loads an LKPack and sets it as the active LKPack

Parameters:

  • name string Name of the LKPack to load, relative to maps/

Usage:

    LK3D.LoadLKPack("deepdive_content")
LK3D.GetDataPathToFile(path)
Gets the real filepath to a LKPack file

Parameters:

  • path string LKPack filepath where to get the real path

Returns:

    string Real path to the file

Usage:

    LK3D.GetDataPathToFile("models/dd_main/items/spray_can.obj")
LK3D.ReadFileFromLKPack(path)
Reads a file from LKPack

Parameters:

  • path string LKPack filepath where to read the file

Returns:

    string Contents of the file, nil if not found

Usage:

    LK3D.ReadFileFromLKPack("models/ponr_main/revolver/revolver_bakeduvs.obj")
LK3D.OpenFileFromLKPack(path, openMode)
Opens a file from LKPack

Parameters:

  • path string LKPack filepath where to open the file
  • openMode string The file open mode, ex. "rb"

Returns:

    file_class File pointer, nil if not found

Usage:

    LK3D.OpenFileFromLKPack("ponr_keyframes/intro/camera.lkf")
LK3D.PlayAudioFromLKPack(path, flags, callback)
Plays a sound file from LKPack

Parameters:

  • path string LKPack filepath to the sound
  • flags string Sound flags, refer to sound.PlayFile
  • callback function Callback to execute once loaded

Usage:

    LK3D.PlayAudioFromLKPack("sound/ponr_main/revolver/revolver_fire.wav")
