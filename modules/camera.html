Module camera

Camera control functions

These functions handle moving / rotating the camera

Functions

LK3D.SetCamPos(pos) Sets the position of the camera
LK3D.SetCamAng(ang) Sets the angle of the camera
LK3D.SetCamPosAng(pos, ang) Sets the position and the angle of the camera
LK3D.SetCamFOV(fov) Sets the FOV of the camera
LK3D.SetCamFarZ(farZ) Sets the far z distance of the camera
LK3D.SetCamNearZ(nearZ) Sets the near z distance of the camera
LK3D.SetCamZDistances(nearZ, farZ) Sets the far z and near z distance of the camera
LK3D.SetCamOrtho(ortho) Enables / disables Orthographic Projection
LK3D.SetCamOrthoParams(params) Sets the ortho parameters of the render


Functions

LK3D.SetCamPos(pos)
Sets the position of the camera

Parameters:

  • pos vector The new position we want to set the camera to

Usage:

    LK3D.SetCamPos(Vector(0, 0, 0))
LK3D.SetCamAng(ang)
Sets the angle of the camera

Parameters:

  • ang angle The new angle we want to set the camera to

Usage:

    LK3D.SetCamAng(Angle(90, 0, 0))
LK3D.SetCamPosAng(pos, ang)
Sets the position and the angle of the camera

Parameters:

  • pos vector The new position we want to set the camera to
  • ang angle The new angle we want to set the camera to

Usage:

    LK3D.SetCamPosAng(Vector(0, 0, 0), Angle(90, 0, 0))
LK3D.SetCamFOV(fov)
Sets the FOV of the camera

Parameters:

  • fov number The new FOV we want to set

Usage:

    LK3D.SetCamFOV(65) -- cinematic :)
LK3D.SetCamFarZ(farZ)
Sets the far z distance of the camera

Parameters:

  • farZ number The new far z we want to set

Usage:

    LK3D.SetCamFarZ(800)
LK3D.SetCamNearZ(nearZ)
Sets the near z distance of the camera

Parameters:

  • nearZ number The new near z we want to set

Usage:

    LK3D.SetCamNearZ(.1)
LK3D.SetCamZDistances(nearZ, farZ)
Sets the far z and near z distance of the camera

Parameters:

  • nearZ optional number The new near z we want to set
  • farZ optional number The new far z we want to set

Usage:

    LK3D.SetCamZDistances(.1, 800)
LK3D.SetCamOrtho(ortho)
Enables / disables Orthographic Projection

Parameters:

  • ortho bool Enable / disable orthographic projection

Usage:

    LK3D.SetCamOrtho(true)
LK3D.SetCamOrthoParams(params)
Sets the ortho parameters of the render

Parameters:

  • params table Ortho parameters, refer to GLua RenderCamData

Usage:

    LK3D.SetCamOrthoParams({
 left   = -.5,
 right  =  .5,
 top    = -.5,
 bottom =  .5,
})
