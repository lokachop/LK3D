LK3D.UpdateRtEz(rt, call)
Render stuff to a rendertarget

Parameters:

  • rt rendertarget Rendertarget to render to
  • call function Function that renders

Usage:

    LK3D.UpdateRtEz(rt_render, function()
  surface.SetDrawColor(255, 0, 0)

  -- ScrW() and ScrH() are the RT size
  surface.DrawRect(0, 0, ScrW() * .5, ScrH() * .5)
end)
LK3D.DeclareTextureFromFunc(index, w, h, func, transp, ignorez)
Declares a LK3D texture from a func

Parameters:

  • index string LK3D texture name
  • w number Texture width
  • h number Texture height
  • func function Function that renders the texture
  • transp bool Whether the texture is transparent or not
  • ignorez bool Whether the texture ignores Z or not

Usage:

    -- extract from internal LK3D
LK3D.DeclareTextureFromFunc("checker", 16, 16, function()
  render.Clear(64, 64, 64, 255)
  surface.SetDrawColor(96, 96, 96)
  surface.DrawRect(0, 0, 8, 8)
  surface.DrawRect(8, 8, 8, 8)
end)

LK3D.GetTexturePixelArray(index, inline)

WARNING

This function is horribly slow!

Gets an array containing the RGB values of the LK3D texture

Parameters:

  • index string LK3D texture name
  • inline optional bool Whether to inline the table, refer to examples

Returns:

    table Texture data

Usage:

  • -- non-inlined
local pixData = LK3D.GetTexturePixelArray("worse_barrel_sheet", false)
local firstPixel = pixData[0][0]
  • -- inlined
local pixData = LK3D.GetTexturePixelArray("worse_barrel_sheet", true)
local firstPixel = pixData[0]

LK3D.ApplyShaderEffect(index, func)

WARNING

This function is horribly slow!

Applies a function to each pixel of the LK3D texture

Parameters:

  • index string LK3D texture name
  • func function Pixel function, refer to usage

Usage:

     LK3D.ApplyShaderEffect("water_bad", function(xc, yc, arr)
	  local cont = arr[xc][yc]
	  local c_data = Color(cont[1], cont[2], cont[3])
	  local h, s, v = ColorToHSV(c_data)

	  surface.SetDrawColor(HSVToColor(h, 0, v))
	  surface.DrawRect(xc, yc, 1, 1) -- grayscale bad
end)
