Module lights

Lets you manage lights

Module to create / update / delete lights on the active universe
It also allows you to get the light intensity at a position on the universe

LK3D.AddLight(index, pos, intensity, col, smooth)
Adds a light to the active universe

Parameters:

  • index string Index name of the light
  • pos vector Position of the light
  • intensity number Intensity of the light
  • col color Colour of the light
  • smooth bool Sets the distance to linear rather than exponential

Usage:

  • -- adds a green light at the center of the universe
 LK3D.AddLight("loka_light_green", Vector(0, 0, 0), 2, Color(0, 255, 0), false)
  • -- adds a huge smooth blue light at the center of the universe
 LK3D.AddLight("loka_light_blue", Vector(0, 0, 0), 3.5, Color(0, 0, 255), true)

LK3D.GetLightIntensity(pos, norm)
Gets the light intensity at a pos on the active universe

Parameters:

  • pos vector Position to get the light intensity from
  • norm optional vector Normal to use to calculate intensity

Returns:

  1. number intR Red channel intensity between 0-1
  2. number intG Green channel intensity between 0-1
  3. number intB Blue channel intensity between 0-1

Usage:

  • -- Gets light values at Vector(0, 0, 0)
 LK3D.GetLightIntensity(Vector(0, 0, 0))
  • -- Gets light values at Vector(0, 0, 0) of a flat plane aiming up
 LK3D.GetLightIntensity(Vector(0, 0, 0), Vector(0, 0, 1))
